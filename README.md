# TechCompass

TechCompass is an open-source personalized career development platform for university students and self-taught developers.

The project helps learners turn scattered free resources into structured, project-based learning paths for tech careers.

## Sprint 2 Focus

Sprint period: 15 May 2026 to 29 May 2026.

- Project charter
- Repository and architecture setup
- Database design
- Web Development sample resource dataset
- Job requirement research
- Sprint review demo preparation
